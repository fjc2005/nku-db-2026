# 数据库工程作业报告模板

## 填写说明

完成一个小型的数据库信息管理系统或部分功能，并填写工程作业报告；程序和报告请在规定时间之内上传。

开发模式可选 B/S 或 C/S，开发高级语言任选，后台数据库使用大型数据库管理系统，如 SQL Server、Oracle、MySQL 等，不使用桌面数据库。

报告中列举的四种操作，每种操作举一个例子即可。作业成绩按照报告中的标准评分，程序只实现报告中涉及的部分即可。

作业完成后，请将工程作业报告和程序打包提交给助教老师，并联系助教老师进行系统说明和演示，回答相关问题。
